# Коммерческое предложение для первых пилотов

## Цифровой гид гостя для отеля

Гость сканирует QR-код в номере, вводит номер комнаты и получает всю важную информацию по проживанию: Wi-Fi, завтрак, заезд и выезд, уборка, услуги, СПА, контакты и ссылки на отзывы.

Сервис не требует установки приложения и может быть запущен как отдельная веб-страница под бренд отеля.

## Какую проблему решает

Для гостей:

- не нужно искать информацию на сайте, в бумажках и у администратора;
- все правила и услуги открываются по QR-коду;
- можно быстро отправить заявку или найти нужный контакт;
- проще оставить отзыв.

Для отеля:

- меньше повторяющихся вопросов на стойке приема и размещения;
- вся информация для гостя собрана в одном месте;
- можно продвигать услуги отеля во время проживания;
- проще направлять довольных гостей на Яндекс и 2ГИС;
- современный сервис без сложной интеграции с PMS/HMS на первом этапе.

## Что входит в пилот

Пилот на 7-14 дней:

- персональная демо-страница под ваш отель;
- структура разделов: Wi-Fi, проживание, еда, услуги, контакты, отзывы;
- общий QR-код для всех номеров;
- ввод номера комнаты после сканирования;
- печатная A6-карточка для номера;
- динамическая QR-карточка для экрана;
- базовая форма заявки на услугу;
- ссылки на отзывы;
- корректировка текстов после обратной связи.
